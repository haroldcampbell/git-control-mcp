# S012: Tool argument passthrough

## Title
Tool argument passthrough

## Spec Summary
Allow git-control tools to accept additional CLI arguments so callers can use more of the underlying git/gh features.

## Spec Goal
Make each git-control tool extensible without adding a new parameter for every CLI flag.

## DoD checklist
- [x] Add optional `extra_args` support to git-control tools
- [x] Document additional arguments in git-control docs and test plan
- [x] Update helper scripts to forward extra arguments
