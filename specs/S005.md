# S005: Git control MCP server

## Title
Git control MCP server

## Spec Summary
Add a second MCP server under `src/git_control` that can stage files, create commits, and open pull requests for a repository inferred from the project root.

## Spec Goal
Enable automated git operations across local repos via MCP tools with safe defaults.

- [x] Run instructions documented from project root
